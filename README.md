# 🌾 AgroMind AI – Smart Farming Assistant

AgroMind AI is a full-stack AI + MLOps-based agricultural assistant that helps farmers with intelligent insights about crops, soil, and plant health.

---

## 🚀 Features

✅ AI-powered farming chatbot (LLM)  
✅ ML-based plant issue detection  
✅ Chat history with MongoDB  
✅ Image upload interface  
✅ Clean React UI  
✅ Docker-based DevOps architecture  
✅ MLOps pipeline (training + prediction + monitoring)

---

## 🧠 Tech Stack

- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB
- AI: Groq LLM API
- MLOps: FastAPI + Scikit-learn
- DevOps: Docker + Docker Compose
